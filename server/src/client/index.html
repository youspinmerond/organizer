This is API for Organizer. You can use it for youself.

Action is planed user deal.

Plan is planed user deal without relate to time.

/api/actions

Node for manipulations with Actions.

/api/actions/:id

Get Action by id.

/api/actions/len

Get length of actions list.

/api/actions/week

Get this week actions.

/api/plans

Node for work with Plans.

/api/plans/:id

Get Plan by id.

/api/plans/len

Get len of plans list.
